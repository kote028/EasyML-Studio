# EasyML-Studio
**NoCodeAI – A Platform to Apply ML and LLM Models Without Coding**

EasyML-Studio is a web-based platform allowing students to experiment with Artificial Intelligence and Machine Learning models visually, without any programming knowledge. 

## Features
- **Realistic 3D Landing Page** integrated with Three.js
- **No-Code Dataset Upload** (CSV preprocessing, visualization)
- **Machine Learning Studio** (Train Scikit-learn Classification/Regression/Clustering models)
- **LLM Playground** (Text Summarization, QA, Sentiment Analysis)

## Tech Stack
- Frontend: React (Vite), TailwindCSS, React-Three-Fiber
- Backend: Python Flask
- Intelligence: Scikit-learn, Pandas, Hugging Face/OpenAI

## Getting Started

### Backend
1. cd backend
2. pip install -r requirements.txt
3. python app.py

### Frontend
1. cd frontend
2. npm install
3. npm run dev
